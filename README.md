### Installation

1. Clone the repositories:
```bash
   git clone https://github.com/aditya-kumar-patraan1/MEETING_APP.git
```  

2. Navigate to the project directory
```bash
cd Client (for Frontend)
cd Server (for Backend)
```
3. Install dependencies

5. Run the app
```bash
npm run dev
```
